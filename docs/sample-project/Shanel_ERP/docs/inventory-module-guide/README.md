# Inventory Module Study Guide

This folder contains page-wise notes for the Inventory and Production module.

## How to use this guide

1. Start with `00-module-overview.md`.
2. Then read pages in the same order you demo in the UI.
3. For each page, follow:
   - Frontend component flow
   - API endpoint(s)
   - Backend controller logic
   - DB tables touched
   - End-to-end data flow
